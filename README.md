# Python Practice Projects

## Description

This repository contains several small Python projects created for practice and learning purposes. These projects demonstrate basic programming concepts such as loops, functions, random number generation, file handling, and user input processing.

## Projects Included

### 1. Math Quiz Game

A command-line game that generates random math questions using addition, subtraction, and multiplication. The user must answer within a limited time, and the program keeps track of the score.

### 2. Contact Manager

A simple contact management system that allows users to add, search, view, and save contacts. The contacts are stored in a JSON file for persistent storage.

### 3. Guess the Number

A game where the computer selects a random number within a user-defined range, and the player must guess it within a limited number of attempts.

### 4. Hangman Game

A classic word guessing game where the player guesses letters to reveal a hidden name before running out of attempts.

## Technologies

* Python
* JSON
* Random module
* Time module

## Purpose

The purpose of this repository is to practice Python programming and improve problem-solving skills through small projects.
